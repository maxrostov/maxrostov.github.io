Instant Translate SidePanel

  1. Распаковать zip-архив
  2. В Настройках расширений включить режим разработчика
  3. Кнопкой "Загрузить распакованное / Load unpacked" выбрать папку
    4. После установки расширения старые (уже открытые) страницы надо перезагрузить (если надр на них делать перевод)

Instant Translate SidePanel is a Chrome extension that allows you to translate text on any website.
